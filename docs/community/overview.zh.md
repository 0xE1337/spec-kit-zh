# 社区

Spec Kit 社区构建了扩展、预设、套装、实战演练和配套项目，不断拓展规范驱动开发能做到的事。所有社区贡献均由各自的作者独立创建和维护。

## 扩展

扩展为 Spec Kit 增加新能力——领域专属命令、外部工具集成、质量关卡等等。目前有来自 70+ 位作者的 130 多个社区扩展，覆盖从无障碍治理到多智能体编排的方方面面。

[浏览社区扩展 →](extensions.md)

## 预设

预设定制 Spec Kit 的行为——覆盖模板、命令和术语，而不改动任何工具。社区预设从语言本地化到完全不同的开发方法论，应有尽有。

[浏览社区预设 →](presets.md)

## 套装

套装把扩展、预设、工作流和步骤组合成面向角色或团队的组件栈，可以一并安装。

[浏览社区套装 →](bundles.md)

## 实战演练

分步指南，展示规范驱动开发在不同场景、语言和框架中的实际运用。

[浏览社区实战演练 →](walkthroughs.md)

## 友邻项目

扩展、可视化或构建于 Spec Kit 之上的社区项目——包括 VS Code 扩展、Claude Code 插件等等。

[浏览友邻项目 →](friends.md)
